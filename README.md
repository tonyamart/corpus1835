### Корпус русской поэзии 1835-1840 гг.

В репозитории хранятся данные для подготовки корпуса русской поэзии второй половины 1830-х гг.

#### Метаданные
Таблица **poetry_collections_database.tsv** содержит данные о сборниках лирических стихотворений, краткую информацию об их авторах и месте издания.

В таблице **poems_publised_in_periodicals.tsv** помещена роспись стихотворений, опубликованных в периодических изданиях 1835-1840 гг.

Во всех таблицах в случае отсутствия данных в ячейке используется обозначение *NA*.

#### База данных отзывов
Таблица **collections_reviews.tsv** содержит библиографические данные об отзывах на сборники, приведенные в таблице **poetry_collections_database.tsv**. 

Корпус отзывов (каждый отзыв в формате .PDF) доступен [по ссылке (.zip-архив)](https://drive.google.com/file/d/18XubGmlqG9Yi5H2omgdQZ-pWVV8-sE1K/view?usp=sharing) (внимание, размер архива -- около 500 МБ!). Отзывы соотносятся со сборниками по идентификационному номеру сборника (COL_ID), указанному в имени каждого файла.
